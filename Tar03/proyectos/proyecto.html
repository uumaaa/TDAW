Demostracion de etiquetas

dl, dt, dd

¿Para que sirve la etiqueta dl?
Se utiliza para representar listas de descripciones, similares a listas de preguntas y respuestas
¿Como se utiliza?
dl es la que representa la lista en general (como un ol), dt es un nombre y dd es su valor
¿Hay limite de nombres (dt) y de valores (dd)?
No, un mismo nombre (dt) puede tener varios valores (dd) y viceversa

blockquote

Sirve para citar pero en forma de bloque, esto quiere decir que se utiliza principalmente

para citar textos acompañados de alguna imagen u otro elemento

tiene un atributo cite que sirve para exponer el link de referencia

q

Similar a blockquote pero exclusivamente para lineas de texto

Cite

kbd y samp

kbd muestra alguna entrada que el usuario puede hacer (generalmente desde el teclado)

Presiona K y despues O para ganar

Mientras que la etiqueta samp hace referencia a una salida, generalmente por alguna accion hecha por el usuario

La computadora generó un error, ERROR 404: NOT FOUND

kbd contiene samp: representa una entrada del usuario generada por la salida la computadora

samp contiene kbd representa una salida que contiene alguna entrada del usuario
